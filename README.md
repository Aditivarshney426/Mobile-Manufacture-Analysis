##📱 Mobile Manufacturing Data Analysis using SQL

📌 Project Overview
This project analyzes mobile manufacturing and sales data to uncover trends in customer purchases, pricing, and brand performance.

---

 📊 Dataset Information
The dataset includes:
- Customer Data
- Location Data
- Manufacturer Data
- Model Data
- Transactions Data

---

 ❓ Key Business Questions Solved
- States with highest mobile purchases
- Top-performing brands like Samsung
- Cheapest mobile phone available
- Average price analysis for top manufacturers
- Customer spending patterns
- Year-wise sales comparison



## 🛠 Tools Used
- SQL Server

---

## 📈 Key Learnings
- Data analysis using SQL
- Multi-table joins
- Business insights generation
- Trend analysis
